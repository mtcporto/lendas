# **App Name**: Lendas

## Core Features:

- Myth List: List all available myths and legends of Brazil.
- Myth Details: Retrieve a myth's data by name.
- Display Myth: Display relevant image and associated detail for each Myth or Legend

## Style Guidelines:

- Primary color: Saturated orange (#E67700) to reflect Brazilian folklore's warmth and energy.
- Background color: Light, desaturated orange (#F2D3B3) for a warm, inviting feel.
- Accent color: Yellow (#D9A628), an analogous color, but adjusted in brightness and saturation, for highlights and interactive elements.
- Headline font: 'Belleza' (sans-serif) for its fashionable, art-oriented look.
- Body font: 'Alegreya' (serif) to support longer blocks of text.
- Use hand-drawn style icons to represent each myth, adding a touch of authenticity.
- Clean and spacious layout with a focus on readability and visual storytelling.
- Subtle transition animations to enhance user engagement without being distracting.
